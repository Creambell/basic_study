D드라이브에 basic_study 파일 생성

settings.gradle 에서 이름을 basic_study로 바꿔줘야 
.jar이 들어있는 Project and External Dependencies가 생깁니다.

Git Bash를 열고 해당 디렉토리로 이동합니다:
cd /d/basic_study/basic_study

이 디렉토리가 Git 저장소가 아니라면 초기화합니다:
git init

원격 저장소를 추가합니다 (이미 설정되어 있다면 이 단계는 건너뛰세요):
git remote add origin https://github.com/Creambell/final-study.git

모든 파일을 스테이징 영역에 추가합니다:
git add .

변경사항을 커밋합니다:
git commit -m "Initial commit of basic_study project"

원격 저장소의 내용을 가져와 병합합니다 (기존 저장소에 내용이 있는 경우):
git pull origin main --allow-unrelated-histories
(충돌이 발생하면 수동으로 해결해야 합니다)
변경사항을 원격 저장소로 푸시합니다:
git push -u origin main
